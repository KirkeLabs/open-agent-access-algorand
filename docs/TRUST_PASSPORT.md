# Trust Passports

A trust passport is a structured publishing artifact that makes a page, claim, or campaign asset legible to people, agents, search systems, and auditors.

An OAA trust passport should include:

- canonical URL
- author or institution identity
- publication and update dates
- claims and source URLs
- AI-use posture
- attribution rules
- agent access policy reference
- mandate expectations for private, paid, or tool-mediated actions
- receipt requirements
- disclosure fields for synthetic or AI-assisted content

The example in `examples/trust-passport-publisher` shows how a publisher can combine `agent-access.json`, `agent-mandates.json`, and `trust-passport.json`.
