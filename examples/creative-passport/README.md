# Creative Passport Example

This example shows how Open Agent Access can model a creative asset such as a
sample, loop, stem, vocal, or dataset.

The passport is intentionally careful:

- it records a rights claim and license terms
- it can generate an OAA access policy
- it can bind provenance references such as C2PA manifest hashes
- it can bind licensing references such as RSL license URLs
- it can bind Algorand receipt/anchor references
- it does not claim to create copyright or prove final legal ownership

Use this pattern for community pilots such as a Manchester sample/stem licensing
passport.
